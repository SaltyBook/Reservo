# Reservo

Reservo ist eine WPF-Anwendung zur Verwaltung von Mieter-Daten und zum Erstellen von Reservierungen und Rechnungen.

Die Anwendung ermöglicht das Laden, Bearbeiten und Speichern von Reservierungsdaten aus Excel-Dateien sowie die Erstellung von Dokumenten und Feedback über externe Services.

## Features

- 📊 Laden und Verwalten von Excel-Dateien (.xlsx)
- 🧾 Verwaltung von Mieter-Daten
- 📅 automatische Erkennung von Datumsüberschneidungen
- 📈 Statistik (Einträge, vergangene Einträge, Stornierungen)
- 🧾 Erstellung von Reservierungen und Rechnungen
- 📬 Feedback-System über Trello
- 🔐 Speicherung von Credentials (lokal & extern)
- 🧩 MVVM-Architektur
- 📝 Logging mit Serilog
